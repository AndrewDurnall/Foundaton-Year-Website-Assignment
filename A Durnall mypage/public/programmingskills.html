Programming Skills

This section displays my skills in ECMAscript via two additional pages described below. ECMAScript is a standard script language, developed with the cooperation of Netscape and Microsoft and mainly derived from Netscape's JavaScript, the widely-used scripting language that is used in Web pages to affect how they look or behave for the user.

Theatre Ticket Booking Form

Theatre Tickets

In this example i have used ECMAScript to create an online booking form for tickets to go to the theatre, this allows users to enter their details and choose a show with how many tickets they need and see the results when the Book button is clicked.

Fruit Machine

Fruit Machine

In this example i have used ECMAScript to create a virtual fruit machine with 9 fruits that when credits are applied the user can try to win by matching fruits either all the way across or the two most right slots, the user can then collect any winnings and this is displayed on screen.
